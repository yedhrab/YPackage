# 🔗 YGoogleDrive

## 🎌 Kullanım Notları

* 🔄 Google Drive bağlantılarını dönüştürmek için `ygoogledrive` komutu kullanılır
* 👁️ Ön izleme bağlantılarını direkt indirme bağlantılarına çevirir
* 🆘 Kullanım detayları için `ygoogledrive -h` yazın
