# 👨‍💼 YFileRenamer

🗃️ Editörlerdeki yeniden adlandırma işlemleri gibi, dosya sistemindeki dosya veya dizinleri topluca yeniden adlandırmayı sağlar

## 🎌 Kullanım

- 💡 Detaylı bilgi için `yfilerenamer -h` komutunu kullanabilirsin
